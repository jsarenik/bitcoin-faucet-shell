PlebNet Playground is just a custom Signet network.

The blocks on PlebNet Playground network are not really mined, but rather just signed, at the moment in very exact 10-minute intervals so that the timing resembles the average timing of blocks on the main Bitcoin network.

Do not use Testnet addresses here. Testnet and Signet use the same address format and the coins sent to your Testnet address will need to be redeemed by importing the private key (or sweeping it) on to Signet.

See an example Caddyfile for Caddy2 web server serving also an efficient Tor hidden service.

Git repository of this POSIX-shell reimplementation of bitcoin-faucet is hosted on GitHub.
